💡 The Challenge

The Problem We Solve

Quantum ML researchers face significant obstacles finding quality datasets for their experiments

📄

Scattered Data

Datasets are buried in academic papers, scattered across GitHub repositories, and hidden in vendor-specific tutorials with no central discovery point.

🔀

Inconsistent Formats

No standardized metadata schemas, inconsistent documentation practices, and varying format conventions make datasets hard to compare and use.

⏰

Time Wasted

Researchers spend countless hours hunting for suitable datasets instead of focusing on what matters—advancing quantum machine learning research.

⚡ Platform Features

What QDatasets Offers

The first open-source platform unifying quantum machine learning datasets. Built by researchers, for researchers.

🗂️

Comprehensive Dataset Library

Curated collection of 50+ quantum ML datasets spanning chemistry, optimization, classification, and simulation. Advanced search filters by qubit count, algorithm type, domain, and hardware compatibility help you find exactly what you need in seconds.

✓

Standardized & Verified

Every dataset includes consistent metadata schemas, verified benchmarks, and detailed documentation. Peer-reviewed quality standards ensure reproducibility and reliability. Download with confidence knowing each dataset meets scientific rigor requirements.

🔧

Framework Agnostic

Works seamlessly with Qiskit, Cirq, PennyLane, TensorFlow Quantum, and other major quantum frameworks. Pre-formatted data loaders, automatic format conversion, and ready-to-use code snippets eliminate integration headaches.

🌍

Open Source & Community-Driven

100% free and open source forever. Contribute your own datasets, share benchmarks, and collaborate with researchers worldwide. Community peer reviews, transparent governance, and public roadmap ensure the platform serves everyone's needs.

Quick Start Example Python 
# Install QDatasets CLI
pip install qdatasets

# Download and load a dataset
from qdatasets import load_dataset

# Automatically formatted for your framework
dataset = load_dataset('qm9-molecular', framework='qiskit')

# Ready to use!
X_train, y_train = dataset.get_train_split()
print(f"Loaded {len(X_train)} samples")
